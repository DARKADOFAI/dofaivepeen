# DOFaI ВэПээН

**Стильный, быстрый и безопасный VPN-клиент под Android.** 
Создан для работы с панелью **Remnawave**, использует мощь **Xray-core**.

![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)
![Xray](https://img.shields.io/badge/Xray_Core-1A1A1A?style=for-the-badge&logo=v2ray&logoColor=white)

</div>

## 🌟 Особенности

- 🔒 **Полная совместимость с Remnawave:** Умная подстановка заголовков (`x-hwid`, `x-app-version`, `x-device-os` и др.) для корректной работы.
- 🎨 **Material You:** Дизайн "подстраивается" под цвета ваших обоев, а интерфейс наполнен плавными анимациями.
- 🔄 **Автообновление подписки:** Приложение само поддерживает список серверов в актуальном состоянии (стягивает в фоне каждые 3 часа).
- 🛡️ **Защита на уровне DNS:** Выбор прямо в приложении (Cloudflare, Google, **AdGuard DNS** для блокировки рекламы).
- 📱 **Quick Settings Tile:** Включайте VPN быстро, прямо из шторки Android.
- 🔀 **Сплит-туннелинг:** Исключайте банковские и системные приложения из VPN-тунеля.
- 📦 **Мощное ядро:** Использует Xray-core (оптимизировано под сервис).

## 🚀 Установка

Скачайте свежий APK-файл со страницы [Releases](../../releases/latest) и установите на ваше Android-устройство.

## 🗺️ Планы на будущее

## 📜 О проекте
Этот репозиторий содержит свежие релизы Android-приложения (APK), созданного **эксклюзивно для работы с моим личным VPN сервисом**. 

Клиент построен на базе открытого ядра Xray-core.
